Vismix is the platform for HKU student to search course timeslot and build timetables without considering the time clashes between courses. 

## **Application Preview**
![App Interface](./misc/AppUI.png)


## **Getting Started**

### Development
```
    yarn react-start
```

### Production

```
    yarn build
```
